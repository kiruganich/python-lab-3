=== Алгоритмический мини-пакет (Лабораторная работа №3) ===

Назначение:
Этот пакет реализует базовые алгоритмы сортировки, структуры данных и вспомогательные инструменты
в рамках лабораторной работы №3 по курсу "Алгоритмы и структуры данных".

Функциональность:
1. Вычисление факториала и чисел Фибоначчи (итеративно и рекурсивно).
2. Три алгоритма сортировки: пузырьковая, подсчётом, блочная.
3. Структура данных: Стек (и его расширение MinStack).
4. Генерация тест-кейсов и бенчмарки.
5. Интерактивный командный интерфейс (CLI) для демонстрации всех компонентов.

Интерактивные команды CLI:

Выберите действие, введя соответствующую цифру:

1. Вычислить факториал или Фибоначчи:
   - Выберите тип вычисления (факториал/Фибоначчи, итеративный/рекурсивный).
   - Введите целое число n.
   - Программа вычислит результат или сообщит об ошибке (например, если n отрицательное или слишком большое).

2. Отсортировать список:
   - Введите элементы списка, разделённые пробелом (например, "3 1 4 1 5").
   - Программа определит тип данных (int или float).
   - Выберите доступный алгоритм сортировки для этого типа.
   - Программа выведет отсортированный список или сообщит об ошибке (например, если введены не числа).

3. Работа со стеком:
   - Выберите, создать обычный стек или MinStack.
   - Введите команды для взаимодействия:
     - push <число>: добавить число в стек.
     - pop: удалить и вывести верхний элемент стека.
     - peek: вывести верхний элемент стека без удаления.
     - min: (только для MinStack) вывести минимальный элемент в стеке.
     - is_empty: проверить, пуст ли стек.
     - len: получить размер стека.
     - quit: выйти из режима работы со стеком.

4. Демонстрация сортировок на случайных данных:
   - Введите размер тестового массива.
   - Выберите тип данных (целые или вещественные).
   - Программа сгенерирует случайный массив и применит соответствующие сортировки, показав результаты.

5. Запустить бенчмарк:
   - Программа сгенерирует несколько тестовых массивов (случайный, почти отсортированный и т.д.).
   - Запустит выбранные сортировки на этих массивах.
   - Выведет таблицу времени выполнения для каждой комбинации "алгоритм - массив".

6. Выйти:
   - Завершить работу программы.

Дополнительные аргументы командной строки:
--help, -h: Вывести этот текст справки и завершить работу.

Реализованные алгоритмы:
- factorial, factorial_recursive: Вычисление факториала.
- fibo, fibo_recursive: Вычисление чисел Фибоначчи.
- bubble_sort: Сортировка пузырьком.
- counting_sort: Сортировка подсчётом.
- bucket_sort: Блочная сортировка.
- Stack: Базовый стек на основе list.
- MinStack: Стек с поддержкой получения минимального элемента за O(1).
