# Алгоритмический мини-пакет (Лабораторная работа №3)

### **Цель:** Реализовать пакет алгоритмов сортировки, структур данных и вспомогательных инструментов на Python

### **Функционал:**
- **Математические функции:**
  - `factorial`, `factorial_recursive` — вычисление факториала
  - `fib`, `fib_recursive` — вычисление чисел Фибоначчи
- **Алгоритмы сортировки:**
  - `bubble_sort` — пузырьковая сортировка
  - `counting_sort` — сортировка подсчётом
  - `bucket_sort` — блочная сортировка
- **Структуры данных (Easy):**
  - `Stack` — стек на основе списка
  - `MinStack` — стек с поддержкой получения минимального элемента за O(1)
- **Вспомогательные инструменты (Medium):**
  - Генераторы тест-кейсов (`rand_int_array`, `nearly_sorted`, и др.)
  - Инструменты бенчмаркинга (`timeit_once`, `benchmark_sorts`)
- **Дополнительный функционал:**
  - CLI (Command Line Interface) для интерактивной работы
  - Возможность сохранения отчёта с бенчмарками в файл
### **Особенности:**
- Соблюдение ограничений: запрещено использование `list.sort()` и `sorted()` внутри реализации сортировок
- Поддержка нормализации в `bucket_sort` для значений вне диапазона [0, 1)
- Обработка исключений (`ValueError`, `IndexError`) в структурах данных
- Интерактивный CLI с меню и командами для всех компонентов
- Покрытие основного функционала тестами `pytest`
- Поддержка аргумента командной строки `--help` или `-h` для вывода инструкции

## Установка

### 1. Клонируйте репозиторий:
```bash
git clone https://github.com/kiruganich/python-lab-3.git
cd python-lab-3
```

### 2. Установите зависимости:

```bash
pip install -r requirements.txt
```

### 3. Запустите программу:

```bash
python main.py
```

## Примеры работы
```bash
=== Алгоритмический мини-пакет ===
Доступные действия:
1. Вычислить факториал или Фибоначчи
2. Отсортировать список
3. Работа со стеком
4. Демонстрация сортировок на случайных данных
5. Запустить бенчмарк
6. Сохранить отчёт с бенчмарками в файл
7. Выйти

Выберите действие (1-7): 1
--- Вычисление факториала или Фибоначчи ---
1. Факториал (итеративный)
2. Факториал (рекурсивный)
3. Число Фибоначчи (итеративное)
4. Число Фибоначчи (рекурсивное)
Выберите (1-4): 3
Введите целое число n: 10
F(10) = 55
```
